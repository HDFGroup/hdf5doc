Parameters:
H5D_alloc_time_t alloc_time      IN: Dataset storage space allocation time.

Valid values are as follows:
H5D_ALLOC_TIME_DEFAULT   Allocates dataset storage space at the default time.
(Defaults differ by storage method.)
H5D_ALLOC_TIME_EARLY Allocates all space when the dataset is created.
(Default for compact datasets.)
H5D_ALLOC_TIME_INCR   Allocates space incrementally, as data is written to the dataset.
(Default for chunked storage datasets.)
  • Chunked datasets: Storage space allocation for each chunk is deferred until data is written to the chunk.
  • Contiguous datasets: Incremental storage space allocation for contiguous data is treated as late allocation.
  • Compact datasets: Incremental allocation is not allowed with compact datasets; H5Pset_alloc_time will return an error.
H5D_ALLOC_TIME_LATE Allocates all space when data is first written to the dataset.
(Default for contiguous datasets.)

The H5D_alloc_time_t ENUM is fully described in H5D_alloc_time_t.html.
