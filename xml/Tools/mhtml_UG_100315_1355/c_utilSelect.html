Selecting the right tool

Extracting data: h5dump or h5ls?

Several command-line utilities can be used to extract data or metadata from a file. The following information can guide you in selecting the most appropriate tool.

All Command-line Utilities

All of the HDF5 command-line utilities:
  • Are scriptable; they can be embedded in scripts to perform one step in a multi-step process.
  • Are free-standing; they can be used without invoking the HDF5 library.
