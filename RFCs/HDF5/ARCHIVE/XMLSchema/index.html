XML Upgrade

The HDF XML DTD and tools have been unchanged for several releases.

It is necessary to update the DTD (the grammar) for HDF5-1.6, and there are a number of desired new features.

The document:
  "XML Upgrade for h5dump"
   http://hdf.ncsa.uiuc.edu/RFC/ARCHIVE/XMLSchema/xml-upgrade-details.pdf
   http://hdf.ncsa.uiuc.edu/RFC/ARCHIVE/XMLSchema/xml-upgrade-details.doc

describes the changes to the DTD, the new Schema, and the changes to the h5dump utility.
