Proposed Changes to the h5dump utility

This document shows a revised proposal for changes to the h5dump utility.

The following tables show

  1. The changes that will definitely be implemented for HDF5-1.6.3

  2. Changes that will be done for 5-1.6.3, time permitting.  These will likely not be done in time for 1.6.3.

  3. Changes that will not be attempted for 1.6.3 unless everything else is done.


In this list, there are at least 5 changes to the DDL.

The basic milestones will be:

  1. RFC for changes to the DDL
  2. RFC for feature changes (after DDL is agreed.)
  3. implement changes (1.6 and 1.7 branch)
  4. implement new tests
  5. check in of all features
