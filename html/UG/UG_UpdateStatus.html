HDF5 User’s Guide Update Status

Last modified: 15 October 2010

Notes for internal review

The HDF5 User’s Guide is being updated to describe HDF5 Release 1.8.x. Highlights to date include:
  • Scope
    • All chapters in sections I and II have been updated.
       
  • Functions and macros
    • C and Fortran functions that have been added to the library in the 1.8.x series have been added.
    • Compatibility macros have been added.
    • Deprecated functions have been removed.
    • Sample code has been revised to account for changed, new, and deprecated functions.
  • Captions
    • Captions have been added or expanded.
  • Editing and format
    • Editing and format are now more consistent across chapters.

 

Additional work: We will be soliciting input in the very near future regarding the next set of priorities. The following items offer a hint of the range of possibilities:

  • Add new chapters discussing objects, links, property lists, and performance.
  • Add more examples.
  • Replace current examples with examples drawn from tested code.
  • Fix various bugs.
  • Perform full, chapter-by-chapter editorial review.

 

We welcome feedback on the documentation and will address requests as resources allow. Please send your comments to docs@hdfgroup.org.
