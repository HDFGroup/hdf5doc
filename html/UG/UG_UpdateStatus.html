HDF5 User’s Guide Update Status


The HDF5 User’s Guide has been updated for HDF5 Release 1.8.11. These updates have been made since the 1.8.10 version of this document was published in November 2012.

The main updates are the following:

  • The direct chunk write API was added. See the “Using the Direct Chunk Write Function” topic in the “Additional Resources” chapter.
  • The H5Dgather/scatter functions were added to the function listing in the “Dataset Function Listings” section in the “HDF5 Datasets” chapter.
  • The H5Pget/set_dxpl_multi functions were removed from the “HDF5 Datasets” chapter. The intended functionality for the functions was never fully implemented, and the functions have always been fundamentally broken. Removing these functions does not affect setting the Multi virutal file driver (VFD) or any expected VFD functionality. Multi VFD usage remains unchanged.
  • An additional explanation for converting enumerations was added to the “Data Transfer: Datatype Conversion and Selection” section in the “HDF5 Datatypes” chapter.

We welcome feedback on the documentation. Please send your comments to docs@hdfgroup.org.
