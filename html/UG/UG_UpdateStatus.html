HDF5 User’s Guide Update Status


The HDF5 User’s Guide has been updated for HDF5 Release 1.8.9. These updates have been made since the 1.8.8 version of this document was published in November 2011.

The main updates are the following:

  • Two topics were added to the “Additional Resources” chapter: “HDF5 File Image Operations” and “Copying Committed Datatypes with H5Ocopy”.
  • In the “HDF5 Datatypes” chapter, the “Strings” section, 6.5.1., was updated.
  • In the “HDF5 Attributes” chapter, sample code was added to the “Large Attributes Stored in Dense Attribute Storage” sub-section. See section 8.5.

We welcome feedback on the documentation. Please send your comments to docs@hdfgroup.org.
