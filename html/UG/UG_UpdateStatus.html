HDF5 User’s Guide Update Status


The HDF5 User’s Guide has been updated for HDF5 Release 1.8.7. These updates have been made since the 1.8.6 version of this document was published in February 2011.

The main updates are the following:

  • The external link cache functions were added. These new functions are listed in the “HDF5 Groups” chapter.
  • Two macros that enable compile-time inclusion or exclusion of code were added. See “The HDF5 File” chapter.

We welcome feedback on the documentation. Please send your comments to docs@hdfgroup.org.
