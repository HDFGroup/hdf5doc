Name: H5Pset_jnl_config
Signature:
herr_t H5Pset_jnl_config( hid_t fapl_id, H5AC_jnl_config_t *config_ptr )

Purpose:
Sets initial journaling configuration.

Description:
H5Pset_jnl_config sets the initial journaling configuration in the file access property list fapl_id according to the values supplied in the buffer specified by config_ptr. This configuration is used when a file is opened with metadata journaling enabled.

H5Pset_jnl_config will fail if an invalid configuration is detected.

Parameters:
hid_t fapl_id    IN: File access property list identifier.
H5AC_jnl_config_t *config_ptr   IN: Pointer to the instance of H5AC_jnl_config_t containing the desired configuration.

The H5AC_jnl_config_t struct is described in H5Fset_jnl_config.

Returns:
Returns a non-negative value if successful; otherwise returns a negative value.
