Name: H5Fget_jnl_config
Signature:
herr_t H5Fget_jnl_config( hid_t file_id, H5AC_jnl_config_t *config_ptr )

Purpose:
Gets current metadata journaling configuration for target file.

Motivation:
H5Fget_jnl_config is part of an HDF5 feature set provided to guard against and recover from metadata corruption.

This feature set is discussed in “Using HDF5’s Metadata Journaling and File Recovery Features” and “Metadata Journaling to Improve Crash Survivability.”

Description:
H5Fget_jnl_config retrieves the current metadata journaling configuration in the instance of H5AC_jnl_config_t pointed to by the config_ptr parameter.

Parameters:
hid_t file_id    IN: File identifier of the target file.
H5AC_jnl_config_t *config_ptr   IN/OUT: Pointer to the instance of H5AC_jnl_config_t in which the current journaling configuration is to be reported.

Initialize the field version to the value of H5AC2__CURR_JNL_CONFIG_VER (defined in H5ACpublic.h).

Values returned in other fields are interpreted as described in H5AC_jnl_config_t.

Returns:
Returns a non-negative value if successful; otherwise returns a negative value.
