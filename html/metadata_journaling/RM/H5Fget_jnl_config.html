Name: H5Fget_jnl_config
Signature:
herr_t H5Fget_jnl_config( hid_t file_id, H5AC_jnl_config_t *config_ptr )

Purpose:
Gets current jounaling configuration for target file.

Description:
H5Fget_jnl_config loads the current journaling configuration into the instance of H5AC_jnl_config_t pointed to by the config_ptr parameter.

Note that the version field of *config_ptr must be initialized, allowing the HDF5 Library to support old versions of the H5AC_jnl_config_t struct. Values in the remaining struct fields are returned by the H5Fget_jnl_config call.

Parameters:
hid_t file_id    IN: File identifier of the target file.
H5AC_jnl_config_t *config_ptr   IN/OUT: Pointer to the instance of H5AC_jnl_config_t in which the current journaling configuration is to be reported.

The H5AC_jnl_config_t struct is described in H5Fset_jnl_config.

Returns:
Returns a non-negative value if successful; otherwise returns a negative value.
