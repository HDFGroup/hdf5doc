Name: H5Pget_jnl_config
Signature:
herr_t H5Pget_jnl_config( hid_t fapl_id, H5AC_jnl_config_t *config_ptr )

Purpose:
Retrieves initial journaling configuration.

Description:
H5Pget_jnl_config retrieves the initial journaling configuration as defined in the file access property list fapl_id. This configuration is returned in the buffer indicated by config_ptr and is used when a file is opened with metadata journaling enabled.

Note that the version field of *config_ptr must be initialized, allowing the HDF5 Library to support old versions of the H5AC_jnl_config_t struct. Values in the remaining struct fields are provided by the H5Pget_jnl_config call.

Parameters:
hid_t fapl_id    IN: File access property list identifier.
H5AC_jnl_config_t *config_ptr   IN/OUT: Pointer to the instance of H5AC_jnl_config_t in which the jouranling configuration is to be returned.

The H5AC_jnl_config_t struct is described in H5Fset_jnl_config.

Returns:
Returns a non-negative value if successful; otherwise returns a negative value.
