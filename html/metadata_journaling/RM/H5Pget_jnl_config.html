Name: H5Pget_jnl_config
Signature:
herr_t H5Pget_jnl_config( hid_t fapl_id, H5AC_jnl_config_t *config_ptr )

Purpose:
Retrieves initial metadata journaling configuration.

Motivation:
H5Pget_jnl_config is part of an HDF5 feature set provided to guard against and recover from metadata corruption.

This feature set is discussed in “Using HDF5’s Metadata Journaling and File Recovery Features” and “Metadata Journaling to Improve Crash Survivability.”

Description:
H5Pget_jnl_config retrieves the metadata journaling configuration as defined in the file access property list fapl_id. This configuration is returned in the buffer indicated by config_ptr and is used when a file is opened with metadata journaling enabled.

Parameters:
hid_t fapl_id    IN: File access property list identifier.
H5AC_jnl_config_t *config_ptr   IN/OUT: Pointer to the instance of H5AC_jnl_config_t in which the journaling configuration is to be returned.

Initialize the field version to the value of H5AC2__CURR_JNL_CONFIG_VER (defined in H5ACpublic.h).

Values returned in other fields are interpreted as described in H5AC_jnl_config_t.

Returns:
Returns a non-negative value if successful; otherwise returns a negative value.
