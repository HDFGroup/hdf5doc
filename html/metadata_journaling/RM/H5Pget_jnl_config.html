Name: H5Pget_jnl_config
Signature:
herr_t H5Pget_jnl_config( hid_t fapl_id, H5AC_jnl_config_t *config_ptr )

Purpose:
Retrieves initial metadata journaling configuration from a property list.

Motivation:
This function is part of HDF5’s metadata journaling feature, which guards against metadata corruption and provides a recovery capability.

Metadata journaling is discussed more fully in “Using HDF5’s Metadata Journaling and File Recovery Features” and “Metadata Journaling to Improve Crash Survivability.”

Description:
H5Pget_jnl_config retrieves the metadata journaling configuration as defined in the file access property list fapl_id. This configuration is returned in the buffer indicated by config_ptr.

This function is used when a file is to be opened or has been opened with metadata journaling enabled.

Parameters:
hid_t fapl_id      IN: File access property list identifier.
 
H5AC_jnl_config_t *config_ptr
    IN/OUT: Pointer to the instance of the H5AC_jnl_config_t struct in which the journaling configuration is to be returned.

Before passing the pointer in the H5Pget_jnl_config call, initialize the version field of H5AC_jnl_config_t to H5AC2__CURR_JNL_CONFIG_VER, a value set internally by the HDF5 Library.

Values returned in other fields are described in the H5AC_jnl_config_t description.

Returns:
Returns a non-negative value if successful; otherwise returns a negative value.
