Metadata Journaling and File Recovery

Motivation

The metadata journaling and file recovery feature set is provided to to guard against metadata corruption.

HDF5 Library functions govern metadata journaling:

  • H5Fset_jnl_config
  • H5Fget_jnl_config
  • H5Pset_jnl_config
  • H5Pget_jnl_config

If a system crash or similar event subsequently results in abandoning a file without properly closing it, leaving the file in a corrupted state, a scriptable command-line tool can use that journal record to bring the metadta up-to-date:

  • h5recover

Precaution

These features preserve only metadata; different steps must be taken to guard against raw data corruption. See <reference>.
