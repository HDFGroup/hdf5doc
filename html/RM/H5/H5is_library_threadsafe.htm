Name: H5is_library_threadsafe

Signature:
herr_t H5is_library_threadsafe(/*OUT*/ hbool_t is_ts)

Purpose:
Determine if the HDF5 library was built with the thread-safety feature enabled.

Description:
The HDF5 Library, although not internally multithreaded, can optionally be built with a thread-safety feature enabled that protects internal data structures with a mutex. In certain circumstances, it may be useful to determine, at run-time, if the linked HDF5 Library was built with this thread-safety feature enabled.

Parameters:
hbool_t is_ts OUT: whether the library was built with thread-safety.

Returns:
Returns a non-negative value if successful. Otherwise, returns a negative value.

Fortran Interface:
None
