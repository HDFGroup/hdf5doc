Name: H5Dget_chunk_storage_size
Signature:
herr_t H5Dget_chunk_storage_size(hid_t dset_id, const hsize_t*offset, const hsize_t*chunk_nbytes )
Purpose:
Returns the size of an allocated chunk.
Description:
H5Dget_chunk_storage_size returns the size of an allocated chunk.
Parameters:
    hid_t dset_id     IN: Identifier of the dataset
    const hsize_t *offset     IN: Offset
    const hsize_t *chunk_nbytes     IN: Chunk number of bytes
Returns:
Returns a non-negative value if successful; otherwise returns a negative value.
