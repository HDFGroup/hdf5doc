Name: H5Pget_all_coll_metadata_ops

Signature:
herr_t H5Pget_all_coll_metadata_ops( hid_t accpl_id, hbool_t *is_collective )

Purpose:
Retrieves metadata read mode setting.

Description:
H5Pget_all_coll_metadata_ops retrieves the collective metadata read setting from the access property list accpl_id into is_collective.

Parameters:
hid_t accpl_id   IN: File, group, dataset, datatype, link, or attribute access property list identifier
hbool_t *is_collective   OUT: Pointer to a buffer containing the Boolean value indicating whether metadata reads are collective (TRUE) or independent (FALSE)
Default mode: Independent (FALSE)

Returns:
Returns a non-negative value if successful; otherwise returns a negative value.

Fortran Interface: h5pget_all_coll_metadata_ops_f
