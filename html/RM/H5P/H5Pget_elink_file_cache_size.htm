Name: H5Pget_elink_file_cache_size
Signature:
herr_t H5Pget_elink_file_cache_size( hid_t plist_id, unsigned *efc_size )

Purpose:
Retrieves the size of the external link open file cache from the specified file access property list.

Description:
H5Pget_elink_file_cache_size retrieves the size of the cache of open files accessed through external links.

Parameters:
hid_t plist_id      IN: File access property list identifier
unsigned *efc_size   OUT: External link open file cache size.

Returns:
Returns a non-negative value if successful; otherwise returns a negative value.

Example Usage:
The following code retrieves the external file cache size on the file access property list fapl_id into a local variable: 
status = H5Pget_elink_file_cache_size(fapl_id, &efc_size);
