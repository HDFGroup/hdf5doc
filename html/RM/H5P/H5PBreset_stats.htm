Name: H5PBreset_stats

Signature:
herr_t H5PBreset_stats( hid_t file_id )

Purpose:
Resets the page buffer statistics.

Description:

H5PBreset_stats resets the page buffer statistics for a specified file identifier file_id.

Parameters:
hid_t file_id IN: File identifier  
Returns:
Returns a non-negative value if successful. Otherwise returns a negative value.

Fortran Interface:
None
