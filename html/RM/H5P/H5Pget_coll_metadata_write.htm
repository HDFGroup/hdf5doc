Name: H5Pget_coll_metadata_write

Signature:
herr_t H5Pget_coll_metadata_write( hid_t fapl_id, hbool_t *is_collective )

Purpose:
Retrieves metadata write mode setting.

Description:
H5Pget_coll_metadata_write retrieves the collective metadata write setting from the file access property into is_collective.

Parameters:
hid_t fapl_id   IN: File access propery list identifier
hbool_t *is_collective   OUT: Boolean value indicating whether metadata writes are collective (TRUE) or independent (FALSE)
Default mode: Independent (FALSE)

Returns:
Returns a non-negative value if successful; otherwise returns a negative value.

Fortran Interface:
None
