Name: H5Pset_elink_fapl
Signature:
herr_t H5Pset_elink_fapl( hid_t lapl_id, hid_tfapl_id )

Purpose:
Sets the file access property list identifer associated with the link access property list.

Description:
H5Pset_elink_fapl sets the file access property list identifier, fapl_id, to use when the target file of the external link associated with lapl_id is accessed. See also H5Pget_elink_fapl and H5Lcreate_external.
Parameters:
hid_t lapl_id IN: Link access property list identifier
hid_t fapl_id IN: File access property list identifier

Returns:
Returns a non-negative value if successful; otherwise returns a negative value.

Fortran90 Interface:
None.

History:
Release     C
1.9.0 Function introduced in this release.
