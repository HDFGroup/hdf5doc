Name: H5Iis_valid
Signature:
htri_t H5Iis_valid( hid_t id )
Purpose:
Check if an id is valid
Description:
H5Iis_valid checks if the given ID is valid. If the ID is valid it returns TRUE otherwise it returns FALSE. The ID may even be in an invalid range (for example, negative), in which case FALSE is returned. An ID is valid if it is in use by the application.
Parameters:
    hid_t id     IN: Identifier to check
Returns:
Returns TRUE if id is valid, FALSE if invalid, negative on failure.
Fortran90 Interface:
This function is not supported in FORTRAN 90.
