Name: H5PLprepend
Signature:
herr_t H5PLprepend( const char *plugin_path )

Purpose:
Insert a plugin path at the beginning of the HDF5_PLUGIN_PATH environment variable.

Description:
H5PLprepend inserts a plugin path at the beginning of the HDF5_PLUGIN_PATH environment variable, used in dynamically loaded plugin filters.

Parameters:
const char *plugin_path     IN: The environment variable path.

Returns:
Returns a non-negative value if successful; otherwise returns a negative value.

History:
Release     C
1.10.2 Function introduced in this release.
