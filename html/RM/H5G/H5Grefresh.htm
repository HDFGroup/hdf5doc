Name: H5Grefresh
Signature:
herr_t H5Grefresh(hid_t group_id )

Purpose:
Refreshes all buffers associated with a group.

Description:
H5Grefresh causes all buffers associated with a group to be cleared and immediately re-loaded with updated contents from disk.

This function essentially closes the group, evicts all metadata associated with it from the cache, and then re-opens the group. The reopened group is automatically re-registered with the same ID.

Parameters:
    hid_t group_id IN: Identifier of the group to be refreshed.

Returns:
Returns a non-negative value if successful; otherwise returns a negative value.
