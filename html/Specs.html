HDF5 Specifications

I've got all these old HDF5 files and no software to read them!
How will I ever figure this out?

These documents provide detailed specifications for the HDF5 file format and for standard representations of several types of data objects commonly stored in HDF5.

    Palette Specification 		Specification for implementation of images and palettes in HDF5 applications
