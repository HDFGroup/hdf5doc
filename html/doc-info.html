HDF5 Documentation

Release 1.8.16    Release 1.6.10

The HDF Group currently supports only the HDF5 Release 1.8.x series.

This page also provides a link to Release 1.6.x series documentation for users who have not yet been able to upgrade. This series is no longer supported.

In new development, The HDF Group is working toward HDF5 Release 1.10.0. This release is due in 2016 and will soon be entering the alpha and beta test phases.


 

Upcoming Development Release

HDF5 Release 1.10.0_alpha/beta

Release 1.10.0 is due in 2016 and the alpha and beta test cycle will soon commence.

New features will be introduced in stages throughout the test cycle and will be documented at the link below as they appear. Developers are encouraged to test these features in new projects and with existing applications. Feedback to The HDF Group regarding failures, bugs, or strengths will be very much appreciated. Send email to help-at-hdfgroup-dot-org.

The alpha and beta level software should not be used in product development as the behavior and/or interfaces may change.

Current Stable Releases

HDF5 Release 1.8.16

Release 1.8.16 is the current release. New projects are strongly encouraged to use this release as it contains many new features, file optimizations, and performance enhancements.

Release 1.8.16 Documentation (latest release):

    • http://hdfgroup.org/HDF5/doc/

HDF5 Release 1.6.10

Release 1.6.10 was the last release in the HDF5 Release 1.6.x series. This series is no longer supported.

Release 1.6.10 Documentation:

    • http://hdfgroup.org/HDF5/doc1.6/

Note that the Release 1.8.x series includes a means of fully emulating 1.6.x series functionality. Many 1.6.x users will be able to meet their needs, while also taking advantage of 1.8.x performance enhancements, by configuring and installing Release 1.8.x accordingly. See API Compatibility Macros in HDF5 for details.
