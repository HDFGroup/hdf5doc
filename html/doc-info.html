HDF5 Documentation

Release 1.10.0    Release 1.8.17    Release 1.6.10

The HDF Group currently supports two series of releases: HDF5 Release 1.10.x and HDF5 Release 1.8.x.

This page also provides a link to Release 1.6.x series documentation for users who have not yet been able to upgrade. This series is no longer supported.

In new development, The HDF Group has recently released HDF5 Release 1.10.0 and Release 1.10.1 is expected in late 2016.
 

Current Stable Releases

HDF5 Release 1.10.0

Release 1.10.0 is currently the most advanced stable release of HDF5. New projects are strongly encouraged to use this release as it contains many new features, file optimizations, and performance enhancements.

Release 1.10.0 Documentation (new release):

New features in 1.10.0 are documented here:

    • New Features in HDF5 Release 1.10.0

The regular 1.10.0 documentation set is located here. Much of the new feature documentation is included; the rest will be integrated over the next few months:

    • http://hdfgroup.org/HDF5/doc/

HDF5 Release 1.8.17

Release 1.8.17 is the current maintenance release. The 1.8.x release series will be mantained and actively supported for a limited time. While no date has been determined, support for this series will eventually end.

Existing projects are encouraged to migrate to the Release 1.10.x series to take advantage of its new features, file optimizations, and performance enhancements.

Release 1.8.17 Documentation (maintenance release):

    • http://hdfgroup.org/HDF5/doc1.8/

Note that the Release 1.10.x series includes a means of fully emulating 1.8.x series functionality. Many 1.8.x users will be able to meet their needs, while also taking advantage of 1.10.x performance enhancements, by configuring and installing Release 1.10.x accordingly. See API Compatibility Macros in HDF5 for details.

HDF5 Release 1.6.10

Release 1.6.10 was the last release in the HDF5 Release 1.6.x series. This series is no longer supported.

Release 1.6.10 Documentation:

    • http://hdfgroup.org/HDF5/doc1.6/

Note that the Release 1.10.x and 1.8.x series both include a means of emulating 1.6.x series functionality. Many 1.6.x users will be able to meet their needs, while also taking advantage of 1.10.x or 1.8.x performance enhancements, by configuring and installing Release 1.10.x or 1.8.x accordingly. See API Compatibility Macros in HDF5 for details if considering the use of 1.10.x (or here if considering 1.8.x).
