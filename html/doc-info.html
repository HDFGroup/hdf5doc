HDF5 Documentation

Release 1.8.16    Release 1.6.10

The HDF Group currently supports only the HDF5 Release 1.8.x series. This page provides a link to Release 1.6.x series documentation for users who have not yet been able to upgrade.

HDF5 Release 1.8.16

Release 1.8.16 is the current release. New projects are strongly encouraged to use this release as it contains many new features, file optimizations, and performance enhancements.

Release 1.8.16 Documentation (latest release):

    • http://hdfgroup.org/HDF5/doc/

HDF5 Release 1.6.10

Release 1.6.10 was the last release in the HDF5 Release 1.6.x series This series is no longer supported.

Release 1.6.10 Documentation:

    • http://hdfgroup.org/HDF5/doc1.6/

Note that the Release 1.8.x series includes a means of fully emulating 1.6.x series functionality. Many 1.6.x users will be able to meet their needs, while also taking advantage of 1.8.x performance enhancements, by configuring and installing Release 1.8.x accordingly. See API Compatibility Macros in HDF5 for details.
