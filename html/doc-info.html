HDF5 Documentation

Release 1.8.3    Release 1.6.9

The HDF Group currently supports two concurrent releases of HDF5. Since there are substantial differences in these releases, this page enables users to locate the desired documentation set.

HDF5 Release 1.8.3

Release 1.8.3 is the current release. New projects are strongly encouraged to use this release as it contains many new features, file optimizations, and performance enhancements.

Release 1.8.3 Documentation (latest release):

  • http://hdfgroup.org/HDF5/doc/

HDF5 Release 1.6.9

Release 1.6.9 is the latest release in the 1.6.x series. This release is supported for users with existing major projects that cannot be ported to the latest release. Users with tools that have not been ported to the newer release may also need to use this release.

Release 1.6.9 Documentation:

  • http://hdfgroup.org/HDF5/doc1.6/

Note that the 1.8.x release series includes a means of fully emulating 1.6.x series functionality. Many 1.6.x users will be able to meet their needs, while also taking advantage of several of the 1.8.x performance enhancements, by configuring and installing Release 1.8.x accordingly. See API Compatibility Macros in HDF5 for details.
