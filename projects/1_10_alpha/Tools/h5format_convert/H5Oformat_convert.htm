Name: H5Oformat_convert

Signature:
herr_t H5Oformat_convert (hid_t loc_id, const char *objectname, hid_t object_convert_plist)

Purpose:
Converts the format of an object in an HDF5 file.

Motivation:
The types of indexes possible for chunked datasets was expanded in 1.10.x versions of the HDF5 Library. This function call can be used to convert chunked datasets made with one version of the library to the other: 1.10.x to 1.8.x or 1.8.x to 1.10.x.

Description:
This routine is currently not implemented.

H5Oformat_convert converts the object objectname associated with loc_id which will be a file or group identifier. Objects in an HDF5 file are groups, datasets, and committed (named) datatypes.

object_convert_plist is the object conversion property list with properties to determine the type of conversion to be performed. Current properties are the following:

H5O_OBJ_CONVERT_DOWNGRADE Use to downgrade the format of the object from a 1.10 format to a 1.8 format.
H5O_OBJ_CONVERT_UPGRADE Use to upgrade the format of the object from the 1.8 format to a 1.10 format.

Parameters:
hid_t loc_id IN: Location identifier for the object.
const char *objectname IN: The object name.
hid_t object_convert_plist IN: The object conversion property list.

Returns:
Returns a non-negative value if successful; otherwise returns a negative value.

Fortran Interface: None
