Name: H5LDget_dset_dims

Signature:
herr_t H5LDget_dset_dims(hid_t did, hsize_t *cur_dims)

Purpose:
Retrieves the current dimension sizes for a dataset.

Description:
This routine retrieves the current sizes of each dimension of the dastaspace for the dataset did through the parameters cur_dims. It will retrieve dimension sizes for non-NULL cur_dims.

Parameters:
hid_t did IN: The dataset identifier
hsize_t *cur_dims OUT: Pointer to an array for storing the current dataspace dimension sizes of the dataset

Returns:
If successful, returns zero. Otherwise, returns a negative value.

Fortran Interface:
None
